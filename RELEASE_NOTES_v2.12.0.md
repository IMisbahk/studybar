# StudyBar 2.12 — Onboarding & Session Log

Small but important quality-of-life fixes.

## What's new

### Onboarding that actually shows up
The welcome tour no longer uses a sheet inside the menu bar popover (those break on macOS). It renders **inline** when you first launch with no subjects.

**Settings → Show Welcome Tour** lets you replay it anytime.

### Session log — pick what you delete
Timeline → Session log now supports:

- Click to select a session; **⌘-click** for multiple
- **Select All** / **Deselect All** for the current search
- **Delete Selected** with confirmation
- "Delete all under 5 min" is still there as a shortcut

## Upgrade

Download the DMG from [GitHub Releases](https://github.com/IMisbahk/studybar/releases), drag to Applications, and replace the existing app. Your data is unchanged.
