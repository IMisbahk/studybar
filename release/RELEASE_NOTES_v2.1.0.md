# StudyBar 2.1 — Themes & Appearance

Make StudyBar yours. Pick an accent palette, tune the menu bar, and style the floating timer — all local, no account.

## What's new in 2.1

### Color themes
Seven built-in palettes that tint progress rings, goal rings, charts, and highlights:

- **Classic** — system blue
- **Forest** — calm greens
- **Sunset** — warm coral
- **Lavender** — soft purple
- **Ocean** — deep teal
- **Rose** — pink accent
- **Monochrome** — minimal slate

Open **Settings → Themes & Appearance** (menu bar popover or dashboard) to preview and apply.

### Menu bar styles
- **Standard** — icon + countdown (default)
- **Compact** — tighter layout, smaller timer text
- **Minimal** — timer text only while a session is active

### Typography & floating timer
- **Rounded timer digits** — optional rounded font for countdown text
- **Themed floating timer border** — accent-colored border on the floating panel (toggle off for a subtler look)

## Upgrade from 2.0

Download the DMG, drag StudyBar to Applications, and replace the existing app. Your subjects, history, goals, and achievements are unchanged.

Automatic updates will offer 2.1 when you're idle (if enabled in Settings → Updates).
