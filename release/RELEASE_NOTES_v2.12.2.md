# StudyBar 2.12.2 — Achievement pop-ups fixed

Each achievement banner now shows **once** — no more replaying your entire trophy case after deleting sessions or editing history.

## What's fixed

### Achievement unlock banners
- Deleting sessions (or bulk cleanup) no longer re-triggers every achievement you've ever earned
- Gamification rebuild still recalculates XP and unlock state correctly — only **new** achievements get the banner

## Upgrade from 2.12.1

Download the DMG, replace the app in Applications, or use in-app update when idle.
