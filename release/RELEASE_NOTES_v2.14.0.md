# StudyBar 2.14 — UI polish

Small fixes for the idle timer popover and floating HUD.

## What's new in 2.14.0

### Floating timer stays on top
The on-screen HUD now uses a floating window level and re-orders above normal app windows during active sessions. Compact layout widened slightly so pause and fullscreen controls aren't cramped at the edge.

### Subject & topic pickers
Side-by-side pickers in the 300px popover now share space evenly (menu style, flexible width). Add buttons are smaller secondary `+` chips instead of oversized circles.

## Upgrade from 2.13.5

Download the DMG from [GitHub Releases](https://github.com/IMisbahk/studybar/releases/latest), replace the app in Applications, or use in-app update when idle.

Your subjects, history, goals, themes, and achievements are unchanged.
