# StudyBar 2.14.1 — Floating timer fullscreen fix

### Fixed
- **Exit fullscreen** — the minimise button now actually returns the on-screen HUD to the small draggable panel instead of leaving compact UI on a full-screen overlay
- **Stale panel size** — closing the menu bar popover after a session no longer resurrects a screen-sized window from a previous fullscreen state

## Upgrade from 2.14.0

Download the DMG from [GitHub Releases](https://github.com/IMisbahk/studybar/releases/latest), replace the app in Applications, or use in-app update when idle.
